# Contact Details Form

A simple, responsive contact details form built with pure HTML and CSS — no frameworks or JavaScript required.

## Features
- Clean, styled form layout with labeled input fields
- Required field validation (First name, Last name, Email)
- Fields for personal details: name, address, city, zip, phone, mobile, and email
- Simple and readable code structure, good for beginners learning HTML forms

## Tech Used
- HTML5
- CSS3 (inline styling)

## How to Use
1. Clone this repository
2. Open `Format.html` in any web browser
3. Fill in the form fields and click Submit
